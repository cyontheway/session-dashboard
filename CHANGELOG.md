# Changelog

## 3.1.0 (2026-04-26)

- **分页加载**：默认加载 50 条会话，底部"加载更多"逐步追加，大项目秒开
- **模型标签**：每条会话显示所用模型名称，多模型切换标记 `+N`
- **备份功能**：新增 `BACKUP_DEST` 环境变量，备份路径可配置
- **环境变量**：`BACKUP_DIR` 支持（替代硬编码备份目录）
- **清理**：移除所有本机硬编码路径

## 3.0.0

- V3 重构：三栏布局、全文搜索、日期筛选、空会话过滤
- 导出 Markdown、收藏会话、深浅主题切换
- `BACKUP_DIR` 备份源切换

## 2.0.0

- V2 迭代：搜索增强、UI 优化

## 1.0.0

- 初始版本：V1 单栏布局，基础会话浏览
