# Changelog

## 3.2.0 (2026-05-02)

- **命名**："All Sessions" → "All CLI"，明确只有 CLI 项目
- **紧凑布局**：边栏、会话列表、详情面板全部缩小字号和行距，信息密度提升
- **按钮并排**：刷新和备份按钮横向排列，备份文字改为中文
- **All CLI 独占**：All Sessions 不再包含 Cowork 会话（v4 功能）

## 3.1.0 (2026-04-26)

- **分页加载**：默认加载 50 条会话，底部"加载更多"逐步追加，大项目秒开
- **模型标签**：每条会话显示所用模型名称，多模型切换标记 `+N`
- **备份功能**：新增 `BACKUP_DEST` 环境变量，备份路径可配置
- **环境变量**：`BACKUP_DIR` 支持（替代硬编码备份目录）
- **清理**：移除所有本机硬编码路径

## 3.0.0

- V3 重构：三栏布局、全文搜索、日期筛选、空会话过滤
- 导出 Markdown、收藏会话、深浅主题切换
- `BACKUP_DIR` 备份源切换

## 2.0.0

- V2 迭代：搜索增强、UI 优化

## 1.0.0

- 初始版本：V1 单栏布局，基础会话浏览
